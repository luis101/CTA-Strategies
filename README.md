# CTA Strategies Framework

This repository contains a modular framework for implementing and backtesting classical CTA-style Futures and FX strategies

## Structure

```text
cta_strategies/
├── __init__.py         
├── backtester.py       # Backtesting engine
├── base.py             # Base classes for strategies and data
├── data.py             # Data loading, processing, and management
├── example.py          # Example script demonstrating how to use the framework
├── risk.py             # Risk management and position sizing logic
├── signals.py          # Trading signal generation
├── strategies.py       # Implementation of specific trading strategies
└── equity_values.png   # Visual output of strategy performance (example equity curve)
```
